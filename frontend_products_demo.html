What happens:

  • Products are filtered by category and search terms
  • Products come from MySQL database via Flask backend

üîß Technical Details

  • Backend API: GET /api/products?page=1&limit=12
  • Response format: JSON with products array
  • Frontend rendering: JavaScript creates HTML cards dynamically
  • Pagination: 12 products per page
  • Images: Loaded from URLs, fallback to category icons
  • Categories: Microcontrollers, Sensors, Actuators, Displays, etc.
